Personality Analyzer

I have a nice homepage that i built with html and CSS. I used an online website to find the colour pallete for the application.

I built this application first by asking question and finding what type of person you are in terms of personality by asking some questions to you.

Some of the styling elements of this project i had some doubts so i learnt from chatgpt just for a couple of CSS Syntax and how to do the gridding + lines on left of boxes which looks cool.

I have multiple graphs which breakdown calculated data clearly, gives you travel insights, recomendations etc.

There are buttons which link to external services like skyscanner, whatsapp, calendar and booking.com.

None of them use the API with API Keys and stuff. I used URL manipulation to achieve this functionality.

By doing this i simplified the application and sticking to functionality instead of working with API's

I really do think API's are better and more scalable and are more flexible so i might just move to API's sooner or later.

I have a budget calculator which calculates budget as per the personality.

There are travel tips and fun facts.

Share in whatsapp + save as txt option in the end of results page.
